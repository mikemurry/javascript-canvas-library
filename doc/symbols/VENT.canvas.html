<private> <inner> _drawHaystack(ctx, steps, strokeStyle, lineWidth)
Draws a series of similar lines as a batch. More efficient than drawing them individually.
Parameters:
ctx
{Object} The canvas context to draw the lines.
steps
{Object[]} An array of points {x,y}. Each item in the array should contain an array of start and end points.
strokeStyle
{String} The canvas stroke style.
lineWidth
{Number} The width of the stroke.
